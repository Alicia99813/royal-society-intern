Simple Site - The OpenSeadragonViewer (IIIF) Extension

  1. Technical Information
  2. Extensions
  3. OpenSeadragon Viewer

This extension creates an OpenSeaDragon viewer based on IIIF resources, either images[1] or manifests[2]. It is an example of a special page that can be added to the pages.json file.

Summary

Extension Variables

For this extension to work the class variable needs to be set to openseadragon and the file variable needs to be a URL to a live IIIF mainifest or the path to a simple text list of either IIIF manifests or individual info.json URLs for IIIF images, as shown below.

The details included in the content and content right variable will be presented as previously described, however the data included in the named file, in this case openseadragon_list.txt will be processed and displayed in the openseadragon viewer. Please note that a full resolvable URL is required when a single IIIF manifest is used, but a simple filename can be used to reference a local file.

  • [1] For more information about how direct image URLs are constructed see:https://iiif.io/api/image/3.0/#51-image-information-request
  • [2] IIIF manifests are simply a structured list of web based images resources that can be viewed in a IIIF compliant viewer, for more details see https://iiif.io/api/presentation/3.0/#52-manifest

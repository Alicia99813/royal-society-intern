Simple Site - The sub-pages.json file

  1. JSON

As noted on the main JSON page you will need to ensure that your sub-pages.json file is valid JSON, please ensure all double quotes in your added text have a / in front, like this /", also please ensure that you text is all on one line, with no newlines.

The sub-pages.json file is very similar to the pages.json file, so check there for the initial description and documentation. The difference as shown below is that the sub-page object includes three extra fields; parent, class and file, as noted in the figure below two of them, class and file, are only required for Extension Sub-pages, for an example of real code please see the sub-pages.json file.

Each of the values within the object can include simple text information or more complex html, however you must ensure that you follow guidelines included of the main JSON page.

The parent field allows you to nest pages under other pages, starting with one of the main pages, but you can then nest sub-sub-pages under sub-pages and so on if really required.

Breadcrumb Navigation

Sub-pages are not given there own tabs, as the main pages are, they are nested in dropdown menus below their parent main page. To make it easier to navigate back up a series of nested sub pages an additional breadcrumb navigation bar is automatically added to all sub-pages, as shown below.
